# Money Health Score – India 🇮🇳

## Problem
95% of Indians do not have a clear financial plan. Most people do not know whether their savings, expenses, and loans are healthy.

## Solution
Money Health Score is an AI-powered personal finance mentor that gives users a clear score (0–100) and simple, actionable advice in under 5 minutes.

## Features
- Easy data input (income, expenses, savings, loans)
- Transparent money health score
- AI-style advisory tips
- Beginner-friendly and explainable logic

## Tech Stack
- Python
- Streamlit

## How It Works
1. User enters basic financial details
2. Rule-based scoring engine evaluates money health
3. AI advisory layer converts signals into simple guidance

## Impact
- Replaces expensive financial advisors
- Saves time and money for retail users
- Scalable to millions of Indian users

## How to Run
pip install streamlit  
streamlit run app.py
